3.1.4. tft_define_types.py¶

Module author: Joe Kwiatkowski

Take a file containing (X, Y, Z) coordinates, and define collection and generation regions.

3.1.4.1. Command-line usage¶

tft_define_types.py  XYZ  COL_GEN_FRACTION  [OPTIONS]

where XYZ is the file containing (X, Y, Z) data in three columns, and COL_GEN is the fraction of the film along the Z-axis which should generate and collect [1].

3.1.4.2. Options¶

-e¶
This option specifies that site energies are given in the xyz file.
[1]So the total fraction of the film that is a generator or a collector is 2 * COL_GEN.
