3.1.7. tft_plot_pf.py¶

Module author: Joe Kwiatkowski

From ToFeT output files, extract the data necessary to plot Poole-Frenkel plots:

sqrt[F (V/cm)]      mobility (cm^2/V.s)

Notice the units of the field are in (V/cm): this makes the experimentalists happy.

3.1.7.1. Command-line usage¶

tft_plot_pf.py  FILES

where FILES are specified by a glob pattern.
