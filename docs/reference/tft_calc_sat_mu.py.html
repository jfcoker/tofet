3.1.3. tft_calc_sat_mu.py¶

Module author: Joe Kwiatkowski

Calculate field-effect mobilities.

3.1.3.1. Command-line usage¶

tft_calc_sat_mu.py  FILES  -l LENGTH  -w WIDTH  [OPTIONS]

where FILES are ToFeT output files specified with a glob pattern. LENGTH and WIDTH are the length and width of the simulated FET channel, measured in Angstrom.

3.1.3.2. Options¶

debug
Output information useful for debugging.
