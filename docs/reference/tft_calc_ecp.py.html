3.1.2. tft_calc_ecp.py¶

Module author: Joe Kwiatkowski

Calculate the Electrochemical potential for each molecule.

3.1.2.1. Command-line usage¶

where OUTPUT_FILE contains the energies and occupation probabilities of each molecule.
