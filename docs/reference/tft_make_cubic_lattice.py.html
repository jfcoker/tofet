3.1.6. tft_make_cubic_lattice.py¶

Module author: Joe Kwiatkowski

Make a cubic lattice, outputting XYZ and edge files.

3.1.6.1. Command-line usage¶

tft_make_cubic_lattice.py  XMAX  YMAX  ZMAX  COL_GEN  J  DE

where (XMAX, YMAX, ZMAX) are the maximum dimensions of the sample, COL_GEN is the fraction of the film which is used for collection and generation, J is the transfer integral between vertices, and DE is the difference in energies of vertices.

Currently, only supports the format in which energies are placed in the edge file
