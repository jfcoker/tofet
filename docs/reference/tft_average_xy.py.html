3.1.1. tft_average_xy.py¶

Module author: Joe Kwiatkowski

Take some four-dimensional dataset (X, Y, Z, VALUE). Divide the dataset into bins along the Z axis and calculate the average <VALUE> for each bin. Return (Z, <VALUE>), where Z is the lower bound of the bin.

3.1.1.1. Command-line usage¶

tft_average_xy.py  DATASET  BINS

where BINS is the number of bins into which the data is partitioned along the Z-axis.
