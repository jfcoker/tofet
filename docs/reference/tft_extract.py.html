3.1.5. tft_extract.py¶

Module author: Joe Kwiatkowski

Extract data from a ToFeT output file(s). Can return either single values, or entire series of data. There are options to tft_extract.py to extract just about any data set from a ToFeT output file.

For full options and usage type:

tft_extract.py --help

3.1.5.1. Example use¶

Extract temperature and mobility from two files:

tft_extract.py  --temp  --mob_vel  OUTPUT_FILE_1 OUTPUT_FILE_2

Extract the photocurrent transient from all files which match the glob ideal_transient*.out:

tft_extract.py --transient  ideal_transient*.out

3.1.5.2. Assumptions made¶

3.1.5.2.1. When searching for series of data¶

In ToFeT’s output file, series of data must have the following form:

... UNIQUE_HEADER ...
... some description of the columns ...
row_0
row_1
row_2
.
.
.
>

Note the final >, which is necessary to tell ToFeT when the series has finished. The UNIQUE_HEADER (or enough of it to make it unique)

must be specified within this module source.

3.1.5.2.2. When searching for single values¶

Values must have the following form:

... UNIQUE_LINE_BEGINNING ... value

Note that the value must be the last element of the line. The UNIQUE_LINE_BEGINNING (or enough of it to make it unique)

must be specified within this module source.
