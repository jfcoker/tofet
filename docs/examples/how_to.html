2.8. How-to miscellany¶

2.8.1. Apply the electric field in different directions¶

ToFeT always considers the electric field to run along the Z-axis. To apply electric fields in a different direction simply rotate the xyz file file with tft_rotate_coords. Note that the electric field can only be applied perpendicular to one of the faces of the unit cell.

2.8.2. Submit batch jobs to ToFeT¶

Use tft_run_batch.

2.8.3. Extract specific data from large output files¶

Use tft_extract.
