1.5. Input files¶

  • 1.5.1. The ‘xyz’ file
    • 1.5.1.1. Vertex types
    • 1.5.1.2. Energies
    • 1.5.1.3. File format
  • 1.5.2. The ‘edge’ file
  • 1.5.3. The ‘sim’ file

ToFeT takes three input files:

wonderful_morphology.xyz
describing the placement of each molecule in the film.
fast.edge
describing the interactions between neighbouring molecules.
foo.sim
containing all the simulation parameters.

Your input files may have any prefix, but they must end in the correct suffix. Each file contains a number of columns, each of which is delimited by any amount of blank space.
